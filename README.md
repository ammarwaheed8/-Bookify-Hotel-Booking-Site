# 🏨 Bookify

A clean, minimal hotel booking website built for a client — no fluff, just what's needed. The site consists of a **landing page** and a **booking form page**, designed with simplicity and ease of use in mind.

---

## 📋 Overview

Bookify is a front-end client project for a hotel booking firm. The client's requirement was straightforward: a simple, no-fancy website that lets visitors learn about the service and submit a booking request. No complex dashboards, no user accounts — just a clean landing experience and a functional booking form.

## ✅ Features

- **Landing Page** — Introduces the hotel booking service with a clean hero section, brief highlights, and a call-to-action button leading to the booking form
- **Booking Form Page** — Collects guest details, check-in/check-out dates, room preferences, and contact information

---

## ⚠️ Known Non-Working Features

Some features are present in the UI but are **intentionally non-functional** as the project has been delivered to the client in its current scope. These include:

- Form submission (no backend/API connected)
- Date picker validation logic
- Any navigation links beyond the two main pages

These features are placeholders for potential future development if the client chooses to extend the project.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

No frameworks, no dependencies — pure front-end as per the client's requirement.

## 🚀 Getting Started

No build steps needed. Just open `index.html` in a browser:

```bash
# Clone the repo
git clone https://github.com/ammarwaheed8/-Bookify-Hotel-Booking-Site.git

# Open in browser
open index.html
```

Or simply drag and drop `index.html` into your browser.
